Aula 2

Aula 2 | Algoritmos: Dando Ordens para Robôs

PDF da Aula
🚀 Desafio Prático

Objetivo: Testar sua capacidade de dar ordens exatas.

  1. A "Vítima": Escolha uma amiga, um irmão ou um responsável para ser o seu "robô".
  2. A Regra: O robô só pode fazer exatamente o que você disser. Ele não pode usar o cérebro humano para "adivinhar" o que você quer dizer.
  3. Missão: Dê ordens para o robô fazer algo simples, como:
    • Colocar um sapato e amarrar o cadarço.
    • Ou servir um copo de água.
  4. O Aprendizado: Se o seu robô ficar parado ou fizer algo engraçado, parabéns! Você descobriu onde o seu "código" precisa de mais detalhes. Isso se chama Debugar!
Aula 3

Aula 3 | Fluxogramas: Representação Visual e Condicionais

PDF da Aula
🚀 Desafio Prático

Objetivo: Criar um mapa de decisão para uma situação real.

  1. A Missão: Imagine que você quer assistir a um filme no celular.
  2. O Desafio: Desenhe no seu caderno um fluxograma para essa tarefa. Não esqueça de colocar pelo menos uma pergunta (Losango).
    • Sugestão de pergunta: "A internet está ligada?" ou "O celular tem bateria?"
  3. A Regra: Use as formas certas (Retângulos para ações e Losangos para perguntas).
  4. O Aprendizado: Se você conseguiu criar um caminho onde todas as setas levam ao "FIM", você acaba de criar a lógica de um aplicativo de verdade!
